# electoral-battleground-20200730

## Generating the geographic map

Commands for filtering a Natural Earth states/provinces/lakes layer in Mapshaper:

```
filter 'adm0_a3 == "USA"'
proj albersusa
clean
filter-fields postal
simplify 4%
```

SVG export settings

```
svg-data="postal"
```
